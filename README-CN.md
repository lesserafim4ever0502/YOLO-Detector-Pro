# YOLO-Detector-Pro

一款基于 PySide6 开发的专业、现代化的 YOLO 目标检测 GUI 应用程序。

## 项目简介

YOLO-Detector-Pro 是一款功能丰富的 GUI 应用，专为实时目标检测和批量处理而设计。它提供了一个简洁、现代的界面，方便用户使用 YOLO 模型，并在图像、视频、文件夹和实时摄像头流中提供灵活的检测模式。

## 核心特性

- 🖼️ **图像检测** - 对单个图像文件进行高精度检测，支持预览和结果保存。
- 📁 **文件夹批量处理** - 高效处理整个目录的图像，并生成详细的文本报告。
- 📹 **视频检测** - 对视频文件进行实时推理，并**自动将标注后的结果保存**到 `results` 文件夹中。
- 📹 **实时摄像头流** - 从连接的 USB 或内置摄像头进行低延迟检测。
- 📊 **动态数据看板** - 可视化统计数据并实时跟踪检测指标。
- ⚖️ **对比模式** - 对两个不同的 YOLO 模型在相同内容上的表现和结果进行侧并侧评估。
- ⚙️ **高级控制** - 实时微调置信度阈值、IOU 阈值、线宽和推理延迟。

## 技术栈

- **UI 框架**: PySide6 (Qt for Python)
- **AI 引擎**: Ultralytics YOLO (支持 v8/v9/v10/v11)
- **计算机视觉**: OpenCV
- **后端架构**: 多线程处理 (QThread)，确保流畅的 UI 体验。

## 安装指南

### 环境要求
- Python 3.8+
- (可选) 支持 CUDA 的 GPU 以实现更快的推理

### 安装依赖
```bash
pip install PySide6 opencv-python ultralytics numpy
```

## 快速上手

1. 将您的 YOLO 模型（`.pt` 文件）放入 `models/` 文件夹中。
2. 启动应用程序：
```bash
python main.py
```

## 使用手册

### 导航侧边栏
- **图片**: 选择并运行单张图片的检测。
- **文件夹**: 选择一个目录进行批量处理。
- **摄像头**: 开启控制台摄像头的实时检测。
- **数据看板**: 查看近期检测的聚合统计信息。
- **对比模式**: 加载两个模型，并排对比它们的性能和结果。

### 顶部状态栏
- **Classes**: 检测到的唯一对象类别数。
- **Targets**: 当前帧中识别出的目标总数。
- **FPS**: 实时处理速度（每秒帧数）。
- **Model**: 显示当前激活的 YOLO 模型。

### 设置面板 (右侧)
- **模型选择**: 实时切换 `models/` 文件夹中的可用模型。
- **置信度阈值**: 调整最小置信度 (0.01 - 0.99)。
- **IOU 阈值**: 微调非极大值抑制 (NMS) 的重叠度。
- **线宽**: 自定义边界框的粗细。
- **推理延迟**: 为测试或演示目的添加延迟。
- **运行/停止**: 控制检测生命周期。
- **保存结果**: 将当前检测结果导出到 `results/` 目录。
